# dotfiles
## Installation
```bash
  git clone https://github.com/yilunlin/dotfiles/ ~/dotfiles && ~/dotfiles/install
```

### Install VIM plugins
Type the following command in vim:
```
  :PlugInstall
```

### Install TMUX plugins
Enter TMUX and press __`-I__.

### Fonts
Pick a nerd font in fonts dir and install it.
